# Combine SwiftUI MVVM Dependency Injection

A sample code snippet demonstrating how to use Combine and MVVM in a SwiftUI app with GitHubService api get using Dependency Injection.

## Structure

    ```bash
    ProjectRoot/
    ├── Models/
    │   └── User.swift
    ├── ViewModels/
    │   └── UserViewModel.swift
    ├── Views/
    │   ├── UserListView.swift
    │   └── UserDetailView.swift
    └── Services/
        └── UserService.swift
    ```

## Build Tools & Version

    ```bash
    Xcode 15.3
    iOS 15.6+
    ```
